
csexe
=====

个人写的部分C#应用，可执行下载包。

大学写了很多，曾经硬盘坏了一次，部分源码丢失。内容包括游戏，社团管理软件，监控，个人软件等。其中带zip是有多个文件，我打包好了，直接下载即可，也可以查看内部文件；不带zip的表示只有一个文件。

---

**E-Photo.zip** 是一个电子相册播放功能，打开可以选择要播放的文件夹路径和歌曲路径，会自动播放歌曲和切换图片，可简单方便的查看相册。

**音乐播放器.zip** 是自己写的一个音乐播放器，实现了打开歌曲列表，歌曲播放、切换等简单功能。

**LGame.exe** 是自己写的一个小游戏，类似连连看但不是，游戏规则是点击一个块，寻找上下左右最近一个块颜色一致的，然后可以消除，在指定时间内消除越多分数越高，点错了会扣分。

**☆XX 生日快乐☆.exe** 是一个祝福软件，可以修改文件名，然后打开变现实文件名的内容，像雪花一样飘落，是送别人电子礼物的不错的选择。

**Thread_TankWar.exe** 
多线程，绘图实现的坦克大战。
